# Neon Timer

A visually stunning stopwatch, countdown timer & lap tracker with neon aesthetics.

## Features
- Stopwatch with millisecond precision
- Countdown timer with presets (30s to 1h) + custom input
- Lap tracking with best/worst highlighting & stats
- Animated circular progress ring
- Particle effects on lap splits
- 4 neon color themes (cyan, pink, green, gold)
- Sound effects (Web Audio API — no external files)
- Keyboard shortcuts: Space (start/pause), L (lap), R (reset)
- Fully responsive, zero dependencies
